QApplication or QGuiApplication: is the Qt class that manages the main event loop and overall application control.
  It’s responsible for:
    Initializing the application (resources, settings, styles).
    Handling user input events (mouse, keyboard, etc.).
    Managing windows and widgets.
    Running the event loop (exec()), which keeps the app alive until it’s closed.
    Simply: without QApplication, your Qt GUI app can’t start or run properly.

return app.exec(): Starts the event loop of the app and keeps it running until the user closes it. The return value is the app’s exit code.
    An event loop is a cycle that waits for events (mouse clicks, key presses, window updates).
